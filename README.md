# Trader Performance vs Market Sentiment Analysis

This project analyzes how trader behavior, profitability, and risk vary across different
market sentiment phases (Fear, Greed, Neutral, Extreme Greed, Extreme Fear).

## Folder Structure
- notebook_1.ipynb: Main analysis, processing, merging, visualizations.
- notebook_2.ipynb: Additional experimental analysis.
- csv_files/: Contains raw + processed CSV files.
- outputs/: Contains all chart outputs from notebooks.
- ds_report.pdf: Final project report.
